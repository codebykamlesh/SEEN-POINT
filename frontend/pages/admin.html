How to Set Up OneDrive

  1. Go to portal.azure.com → Azure AD → App registrations
  2. Create a new registration (name: SEEN POINT)
  3. Add redirect URI: {backend_url}/api/onedrive/callback
  4. Copy Client ID → set as ONEDRIVE_CLIENT_ID in .env
  5. Create a Client Secret → set as ONEDRIVE_CLIENT_SECRET
  6. API Permissions → Microsoft Graph → Files.Read.All (Delegated)
  7. Click "Connect OneDrive" above to complete the OAuth flow
